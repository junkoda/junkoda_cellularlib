# junkoda_cellularlib
A Code for Kaggle Recursion Cellular Image Classification


## Install

Directory structure assumed is

+ cellularlib
+ input
|   + train
|   |  + HPEG2-01
|   |  + HPEG2-02


```bash
$ make
```

or

```bash
$ pip install -e py
```


## Requirements

* Python3
  - numpy
* C++11 compiler
* Eigen
